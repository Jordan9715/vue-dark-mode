# Nuxt.js

The @vue-a11y/dark-mode is an option for SPA and static apps Vue.js, it does not currently support SSR.

The good news is that Nuxt.js already has an incredible color mode module (@nuxtjs/color-mode) that you can find here: https://github.com/nuxt-community/color-mode-module
