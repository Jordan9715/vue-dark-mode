# Examples

# Simple button

<vue-dark-mode>
  <template v-slot="{ mode }">
    Color mode: {{ mode }}
  </template>
</vue-dark-mode>

Output




# Icon Toggle button

<vue-dark-mode>
  <template v-slot="{ mode }">
    <MyLightIcon v-show="mode === 'light'">
    <MyDarkIcon v-show="mode === 'dark'">
    <MySystemIcon v-show="mode === 'system'">
  </template>
</vue-dark-mode>
